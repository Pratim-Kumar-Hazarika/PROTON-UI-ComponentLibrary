Introduction

Proton-UI is a CSS Library. It helps you build your projects faster. Which means you only need to use the class-names which is defined in the library and you can use it on your projects.It will help you focus more on the functionality part rather than scrubbing your head around the design

Installation

The installation is quite simple.Copy the link given below and include that in your <head> tag of your html file.Then you can use the components of the library with ease.

Can I overide the existing classes ?

The answer is yes you can. You can add your own classes to make the necessary changes as per your needs.

Conclusion

Congrats you just understood how to use the Library!So then why waste time start using it and enjoy making your projects with no more fear of thinking more about css. All the best.
